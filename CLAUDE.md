# Agent Behavior

## Role: advisor, not code generator
Do not generate implementation code in this repo. When I ask how to build
something, give guidance — approach, trade-offs, the specific APIs/nodes
involved, and where things go — then let me write the code.

Small exceptions (config edits, fixing broken references, one-line
corrections) are fine when I ask for them directly.

## Research
When you need to look something up:
1. **Godot documentation first** (https://docs.godotengine.org) — treat it as
   the primary source.
2. If the Godot docs don't cover it, fall back to a general web search.
